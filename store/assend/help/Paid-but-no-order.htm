Paid but no order Instructions

 

In admin go to tools-paid but no order, this will open a new window and will be a webpage in your store.

 

Find the customer and his/her cart click on login, now proceed with the checkout when you get to checkout_shipping.php click on the admin page in the old browser window.

 

Go to Setup-Modules-Payment and enable cash on delivery if you don’t already offer this method. If you don’t offer this and are worried some customer might use this option, set site down for maintenance in tools menu.

 

Now click back to the shop browser window click continue select cash on delivery as payment method click continue, then continue again to checkout_success.php

 

Now click back to admin browser window and remove cash on delivery as a payment method.

 

If you set site down for maintenance then don’t forget to bring site back online

 

The customers order has now been completed, and will be in orders. Go to orders view pending if pending is the default status of your orders, if not go to whatever status is your default status.

 

Find the order you just put through for that customer click on that order, click modify. You can now fully update the order, amend payment from cash on delivery to whatever payment method the customer used, amend any addresses for delivery, billing if not default address.

 

Now this is very important click on tools-security-change basket password and change the password, do this every time you have used the paid but no order feature. This is for security reasons
