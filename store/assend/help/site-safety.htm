Making your site a bit more safe

 

 

Now lets do some web site security.  First we will hide the images directory a bit.  Open catalog/includes/configure.php and find the line;

 

  define('DIR_WS_IMAGES', 'images/');

 

 

Now we will rename the directory to something a little less obvious such as "my-web-images"

 

  define('DIR_WS_IMAGES', 'my-web-images/');

 

Save the file and rename your images folder to your new name also.

 

This will not totally protect your images folder but it will take one way away from hackers finding it.  Reset the permissions to 755 or the same as all your other folders, never keep a folder with permissions of 777.

 

 

**************************************************************

 

Now lets move onto the admin directory.  In admin/includes/configure.php find

 

  define('DIR_WS_ADMIN', '/admin/'); // absolute path required

  define('DIR_FS_ADMIN', '/home/myroot/public_html/admin/'); // absolute pate required

 

 

And change your admin to something REALLY REALLY obscure such as

 

  define('DIR_WS_ADMIN', '/Zz.496.tDxL7/'); // absolute path required

  define('DIR_FS_ADMIN', '/home/myroot/public_html/Zz.496.tDxL7/'); // absolute pate required

 

Save the file and don't forget to rename your admin directory to the same really really obscure name.

 

Open up robots.txt in root folder of your store and add

Disallow: /new_admin_folder_name/

Changing new_admin_folder_name to name you renamed admin too

 

 

**************************************************************

 

Permissions on folders should be no higher than 755. If your hosting setup demands permissions of 777 on folders then change hosts.

.htaccess, botfile.txt and iplog.txt in root folder of your store and sitemonitor_log.txt in admin folder need to be writeable 666 theyre are protected via .hataccess file no one can see them via browser.

 

Now that you have renamed the admin folder, rename the dummy-admin folder to admin this will look like the real admin login page, but is a fake one that does nothing.
