FCK Instructions

Change read and write permissions to 777 for: userfiles in the root of your website.

catalog/userfiles/

catalog/userfiles/file

catalog/userfiles/media

catalog/userfiles/image

catalog/userfiles/flash

Insert the path to your /userfiles/ directory in:

admin/fckeditor/editor/filemanager/connectors/php/config.php

$Config['UserFilesPath'] = '/userfiles/' ;

and a few lines down:

 $Config['UserFilesAbsolutePath'] = '/YOUR/ABSOLUTE/PATH/OF/THE/USERFILES/DIRECTORY/' ;

 Install any Addons in the Addons directory.
