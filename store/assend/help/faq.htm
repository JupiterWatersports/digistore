How do I change site width?

Default store template admin-templates-default template  click on

Change site width, click on modify, change value click update all done

 

If using additional templates goto includes/sts_templates/template name.

Template name being name of the template being used open up the stylesheet.css

look for

#container {

width:950px;

}

 

and amend the width to what you want.

 

If using additional templates goto includes/sts_templates/template name/xxx.php.html

template name being name of the template being used, xxx.php.html being name of files, alter width of columns, change for all files in template name do not go into sub folders

 

find all occurrences

<td width="160" valign="top"><table border="0" width="160" cellspacing="0" cellpadding="2" class="columndiv">

 

change 160 to size you want

 

 

 

How do I remove the right column from a page?

Find in page you want to remove right column from if not using additional templates

Change this

 

<!-- body_text_eof //-->

    <td width="<?php echo BOX_WIDTH; ?>" valign="top"><table border="0" width="<?php echo BOX_WIDTH; ?>" cellspacing="0" cellpadding="2">

<!-- right_navigation //-->

<?php require(DIR_WS_INCLUDES . 'column_right.php'); ?>

<!-- right_navigation_eof //-->

    </table></td>

  </tr>

</table>

<!-- body_eof //-->

 

To this

 

<!-- body_text_eof //-->

            <td width="<?php echo BOX_WIDTH; ?>" valign="top"><table border="0" width="0" cellspacing="0" cellpadding="2">

            </table></td>

          </tr>

        </table>

<!-- body_eof //-->

 

Using additional templates goto includes/sts_templates/template name/xxx.php.html

template name being name of the template being used, xxx.php.html being name of file you wish to remove right column from, if you don’t find a file with that name then you will need to copy one of the others and rename it

 

Find

<td valign="top">

      <p>$content</p>

      <p>&nbsp;</p>

    </td>

    <td valign="top" width="160">

      <table border="0" width="160" cellspacing="0" cellpadding="2" class="columndiv">

        <tr><td>$couponbox</td></tr>

        <tr><td>$cartbox</td></tr>

        <tr><td>$currenciesbox</td></tr>

        <tr><td>$reviewsbox</td></tr>

        <tr><td>$specialbox</td></tr>

        <tr><td>$bestsellersbox</td></tr>

        </table>

    </td>

 

change to this

<td valign="top">

      <p>$content</p>

      <p>&nbsp;</p>

    </td>

How do I add a new box?

 

Admin-setup-infobox admin, follow instructions, each section has a pop up help

 

If using additional templates, check in \includes\modules\sts_inc\sts_column_left.php

 

The box is actually listed if not you will have to add it, new_box.php in this example is the name of the box you are adding

 

Add before the ?>

 

  require(DIR_WS_BOXES . 'new_box.php');

  $sts->restart_capture ('newbox', 'box'); // Get the new box

 

Now to use that in the templates you simply add $ prefix so it becomes $newbox

 

Simply open the files you wish to add it in in, includes/sts_templates/folder name

files in that folder xxxx.php.html look for <tr><td>$currenciesbox</td></tr>

to find where to place it, that would be in the right column, anything before the

<td valign="top">

      <p>$content</p>

 

Will be the left column

 

 

Where is this text located?

 

Download simple search and replace http://www.softpedia.com/get/System/System-Miscellaneous/Simple-Search-Replace.shtml

 

Click on browse after running the software point it to digistore folder make sure replace is not ticked enter text you want to find and click start.

 

In the locations window in results double click the location of the file and it will open up in wordpad

 

How do I add an oscommerce template I bought to Digistore?

 

Follow the instructions for a modified site, includes/functions/general.php & includes/functions/html_output you will have to use a file compare program such as http://winmerge.org/ and compare the files with the template ones and add any extra functions to the template files.Failure to do this, will result in your store not working properly ie like the SEO url’s

 

How do I enable global product quantity discount?

 

Admin-modules-order total and install global product qty discount then click on modify and change discount.

Please note if you have 2 modules with same sort order them 2 modules wont work

How do I add a link in admin?

 

Open admin/advanced_menu.php find what section you want to put link in

Simply copy a link and paste below and amend it to what you want ie we will use customers menu for new link , last link in customers menu is

stm_aix("p5i3","p0i0",[0,"    <?php echo BOX_REPORTS_INACTIVE_USER ?>    ","","",-1,-1,0,"stats_inactive_user.php"]);

stm_aix("p5i4","p1i0",[]);

 

so copy that and paste it below amending so its like after u amended the numbers

 

stm_aix("p5i3","p0i0",[0,"    <?php echo BOX_REPORTS_INACTIVE_USER ?>    ","","",-1,-1,0,"stats_inactive_user.php"]);

stm_aix("p5i4","p0i0",[0,"    <?php echo BOX_REPORTS_INACTIVE_USER ?>    ","","",-1,-1,0,"stats_inactive_user.php"]);

stm_aix("p5i5","p1i0",[]);

 

now insert your link BOX_REPORTS_INACTIVE_USER is what you would have added to your language files, if you only want for 1 language replace

<?php echo BOX_REPORTS_INACTIVE_USER ?> with text ie new link here

 

change the file name to what ever file the link is for

 

If the new link uses configuration.php look for the same number in the sql this will be the id ie
